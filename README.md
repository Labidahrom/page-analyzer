## Project description
Example link: https://python-project-83-production-7904.up.railway.app/
A simple url analyzer. This parser allows you to get the url status code and content of the meta tags:
- title
- h1
- description

## System requirements
- Linux
- Python (3.6 or later)
- PIP
- Poetry
- GIT
- PostgreSQL
